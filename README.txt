این پروژه یک سیستم مسیر‌یابی هوشمند است که با استفاده از
الگوریتم دایکسترا، کوتاه‌ترین مسیر بین دو نقطه روی نقشه
را فقط از طریق خیابان‌های تعریف‌شده محاسبه و نمایش می‌دهد
----------------------------------
- Language: Python 3.8 یا بالاتر
----------------------------------------------
2. کتابخانه‌های مورد نیاز (Requirements)
برای اجرای این پروژه، کتابخانه‌های زیر باید نصب باشند:
1. tkinter
   - برای ساخت رابط گرافیکی (GUI)
   - به‌صورت پیش‌فرض همراه پایتون نصب است
2. ttk (زیرمجموعه tkinter)
   - برای ComboBox و ظاهر گرافیکی بهتر
3. heapq
   - برای پیاده‌سازی صف اولویت‌دار در الگوریتم دایکسترا
   - به‌صورت پیش‌فرض در پایتون وجود دارد
4. math
   - برای محاسبه فاصله اقلیدسی بین نقاط
   - کتابخانه داخلی پایتون
5. Pillow (PIL)
   - برای بارگذاری و نمایش تصویر نقشه
----------------------------------------------

4. فایل‌های پروژه

- map.py
  فایل اصلی پروژه که شامل:
  - پیاده‌سازی گراف
  - الگوریتم دایکسترا
  - رابط گرافیکی (GUI)

- map.png
  تصویر نقشه واقعی که مسیر روی آن نمایش داده می‌شود

- README.txt
  فایل توضیحات پروژه 
----------------------------------------------

6. نحوه استفاده از برنامه

1. برنامه اجرا می‌شود و نقشه نمایش داده می‌شود
2. مبدأ را از ComboBox انتخاب کنید
3. مقصد را از ComboBox انتخاب کنید
4. روی دکمه "Run Dijkstra" کلیک کنید
5. کوتاه‌ترین مسیر با رنگ قرمز نمایش داده می‌شود

------------------------------------
7. توضیح عملکرد الگوریتم

- نقشه به‌صورت یک گراف وزن‌دار مدل‌سازی شده است
- نقاط نشان‌دهنده تقاطع‌ها هستند
- خیابان‌ها به‌صورت دستی تعریف شده‌اند
- وزن هر خیابان برابر طول آن است
- الگوریتم دایکسترا کوتاه‌ترین مسیر را پیدا می‌کند
- مسیر فقط از روی خیابان‌ها عبور می‌کند
